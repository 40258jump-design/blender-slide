# Blender Slide

Blender Slide is a Blender add-on that allows collections to be used as presentation slides.

## Features

- Collection based slides
- Next / Previous navigation
- EXCLUDE mode
- HIDE mode
- Presentation workspace
- Optional shortcuts

## Installation

1. Download blender_slide.py
2. Blender > Edit > Preferences > Add-ons
3. Install from Disk
4. Enable Blender Slide

## Blender Version

4.2+
